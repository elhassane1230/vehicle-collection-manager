# Images

Place the vehicle photos here, named exactly as referenced in the data
(`data/sample-collection.csv`), e.g.:

- `clio_noir.jpg`
- `audi_a7.jpg`
- `yahama_kawazaki.jpg`
- `bmw_moto.jpg`
- `mercedes_actros.jpg`

`pas_image.png` is the fallback shown when an image is missing.
Any missing photo simply falls back to that placeholder — the app still runs.
